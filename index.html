Jo and John's wedding!

How can you help?

When you arrive:

  • Table decorations: Lanterns and wooden stands, flower wreaths, and balloons, 2 of different lengths
  • Top Table decorations: Vase and flowers, flower petals, candles (unlit)
  • Bunting in marquee: Find or ask for ladder from venue, attach using string or sellotape
  • Set up games: Spike Ball: net and balls and rules and Target Practice: net and tennis balls
  • Balloons! For table decorations: helium tank, balloons, string/ribbon, scissors. Trial the length of one table's balloons and find two good lengths of string - cut the string to size. Blow up rest of balloons - not too full, and attach to lanterns on tables using one string of each length.
